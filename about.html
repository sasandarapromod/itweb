Marble වෙබ් අඩවියට ඔබ සියලු දෙන සාදරයෙන් පිලිගන්නවා. සයිබර් අවකාශය තුල Networking, windows Servers, Cloud, Data Center මෙන්න මේ දේවල් පිලිබඳ මුලසිට පිලිවලට සිංහලෙන් ඉගෙන ගන්න තැනක් හොයා හොයා හිටිය අයට Marble කියන්නේ කදිම තෝතැන්නක් මේ දේවල් ඉගෙන ගන්න අයට, මේ ක්ෂේස්ත්රයේ රැකියා කරන අයට හා මේ ක්ෂේස්ත්රයේ රැකියාවක් බලාපොරොත්තු වන අයට මේ සියල්ලන්ටම ගැලපෙන විදිහට මේ ක්ෂේස්ත්රයේ මන් ලබපු අත්දැකීම් තුලින් වගේම මේ විෂයන්ට ප්‍රවීනයන් සමග එක්ව ඔවුන්ගේ අදහස්ද සමගින් තමා මේ අඩවිය නිර්මාණය වෙන්නේ.

සයිබර් අවකාශය තුල Networking, windows Servers, Cloud, Data Center මෙන්න මේ දේවල් Basic concepts වල ඉදල advanced මට්ටමට වෙනකන් වගේම මේ side එකේ එන ගැටළු පිළිබඳවත් කතා කිරීමට බලාපොරොත්තු වෙනවා. ඒ අනුව මුලික වශයෙන් Windows Servers, Microsoft Azure, Microsoft 365, System Center මේ දේවල් පිලිබදව සෑම සතියකම ලිපි පලවෙන අතර CISCO Device Configurations, Cloud Computing, Advanced Networking වැනි මාතෘකා පිළිබඳවත් සෑම මසකම ලිපි පල කිරීමට බලාපොරොත්තු වෙනවා. විශේෂයෙන් මේ අඩවිය Microsoft Technologies වලට focus වෙලා තමා නිර්මාණය වෙන්නේ.
