    Azure

    Microsoft සමාගම විසින් නිර්මනය කරන ලද Microsoft Azure මගින් විවිධ applications සහ සේවාවන් නිර්මාණය කිරීමට, test කිරීමට, deploy කිරීමට හා manage කිරිමට පහසුකම් සපයා ඇත. මෙමගින් Software as Service, අදි බොහෝ සේවාවන් ලබා දේ.

    System Center

    Microsoft ආයතනයේ System Center Suite එක මගින් Systems Management සේවාවන් සපයා දේ. මෙහි විවිධ Software ගණනාවක් ඇත. මෙහි වැඩි වශයෙන් භාවිතා වන්නේ System Center Configuration Manager (SCCM) වේ.

    Windows Server

    Windows server යනු Microsoft සමගම විසින් නිර්මාණය කරන Server Operating System එක්ක වේ. Enterprise Level එකේ Client Computers පාලනය කිරීමට මෙන්ම On Promises Environment එකක විවිධ features හා Roles මෙම Server OS එක මගින් ලබාගත හැක.

    Office 365

    Office 365 යනු Microsoft Office Products වලම cloud base වුනු මුහුණුවරකි. එයට Teams, SharePoint, Exchange Online, Yammer වැනි නව සේවාවන් රැසක් මෙන්ම security අතින්ද වැඩි දියුණු වී ඇත.
